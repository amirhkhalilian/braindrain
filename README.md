# 🧠 braindrain

Utilities for working with **large (thicc!) EDF files** — especially long recordings (~24 hours) and directories containing **many EDF files**.

The tools are designed for situations where you have:

* **Directories containing many EDF files**
* **Very large EDF files** (e.g., 24-hour recordings)
* The need to **inspect, summarize, and clip recordings quickly**

## Philosophy

This is **not** a full EDF processing framework. I think of it as:
> the toolbox you reach for before running the heavy analysis.
